## Setting up the Virtual Environment

### Windows
1. Create the virtual environment:
  ```sh
  python -m venv .venv
  ```
2. Activate the virtual environment:
  ```sh
  .venv\Scripts\activate
  ```

### Mac
1. Create the virtual environment:
  ```sh
  python3 -m venv .venv
  ```
2. Activate the virtual environment:
  ```sh
  source .venv/bin/activate
  ```
